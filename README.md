# google-sheets-docs-sync

# Умна синхронизация Google Таблица ↔ Google Документи

Система за синхронизация, която преобразува редове от Google Таблица в Google Документи с максимална икономия на API лимити.

## Функционалности
- ✅ MD5 хеширане за детекция на промени (спестява 80% от операциите)
- ✅ Групови операции (batch) за четене и запис
- ✅ Поддръжка на "активни" редове (колона F)
- ✅ Цветово кодиране на статусите
- ✅ Обобщен отчет след синхронизация

## Структура на таблицата
| Колона | Име | Описание |
|--------|-----|----------|
| A | Име | Заглавие на документа |
| B | Въпроси | Съдържание за секция "ВЪПРОСИ" |
| C | Съдържание | Съдържание за секция "ОТГОВОРИ" |
| D | URL | Автоматично попълва се с линк към документа |
| E | Статус | Автоматично попълва се със статус |
| F | Активен | TRUE/ДА/1 = обработва се, FALSE/НЕ/0 = пропуска се |
| G | Хеш | Автоматично попълва се с MD5 хеш |

## Изисквания

- Права за редактиране на Drive и Documents
